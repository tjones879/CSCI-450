# HW-3

Due: Oct. 23 2017

Task: Write a simple calculator parser using a recursive descent parser generator.

### Compile and Run

With leg installed in your machines path:

```
leg calc.leg > ex.c
gcc ex.c
```

### Examples

```
cat input.txt && ./a.out < input.txt
1+2+3
5*3-6/4

6.000000
13.500000
```
